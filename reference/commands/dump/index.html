    • -i, --include types

    dump [target...]

    Dump the target state of targets. If no targets are specified, then the entire target state.

    -f, --format json|yaml

    Set the output format.

    -i, --include types

    Only include entries of type types.

    Example

    $ chezmoi dump ~/.bashrc
$ chezmoi dump --format=yaml
